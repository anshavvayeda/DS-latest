**Key Technical Concepts**
-   **Frameworks**: FastAPI (Backend), React (Frontend)
-   **Database**: PostgreSQL
-   **Authentication**: Dual JWT (Cookie + Bearer Token)
-   **AI Orchestration**: Multi-LLM strategy using OpenRouter.
    -   **GPT-4o**: For high-reasoning tasks (subjective rubric evaluation, numerical step-checking).
